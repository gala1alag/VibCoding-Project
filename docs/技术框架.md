## 二、架构设计

### 整体架构

采用 **MVC 分层架构**，将界面、业务逻辑、数据访问三层分离，便于模块独立开发和后期维护。

```
┌─────────────────────────────────────────┐
│               View 层（PyQt5）           │
│   各模块 UI 组件、窗口、对话框            │
├─────────────────────────────────────────┤
│             Controller 层               │
│   业务逻辑、事件处理、模块间协调          │
├─────────────────────────────────────────┤
│              Model 层                   │
│   数据模型、SQLite 操作、文件系统访问     │
└─────────────────────────────────────────┘
```

## 三、数据库设计

数据库文件：`data/devhub.db`（SQLite）

### projects 表
```sql
CREATE TABLE projects (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    name            TEXT NOT NULL,
    path            TEXT NOT NULL,
    type            TEXT NOT NULL CHECK(type IN ('folder', 'file')),
    language        TEXT,
    note            TEXT,
    content         TEXT,
    default_editor  TEXT,
    created_at      TEXT DEFAULT (datetime('now', 'localtime')),
    last_opened_at  TEXT
);
```

### snippets 表
```sql
CREATE TABLE snippets (
    id          INTEGER PRIMARY KEY AUTOINCREMENT,
    title       TEXT NOT NULL,
    language    TEXT,
    code        TEXT,
    description TEXT,
    project_id  INTEGER REFERENCES projects(id) ON DELETE SET NULL,
    created_at  TEXT DEFAULT (datetime('now', 'localtime'))
);
```

### todos 表
```sql
CREATE TABLE todos (
    id          INTEGER PRIMARY KEY AUTOINCREMENT,
    project_id  INTEGER REFERENCES projects(id) ON DELETE CASCADE,
    title       TEXT NOT NULL,
    description TEXT,
    status      TEXT DEFAULT 'todo' CHECK(status IN ('todo', 'doing', 'done')),
    priority    TEXT DEFAULT 'medium' CHECK(priority IN ('high', 'medium', 'low')),
    due_date    TEXT,
    created_at  TEXT DEFAULT (datetime('now', 'localtime'))
);
```

### study_logs 表
```sql
CREATE TABLE study_logs (
    id               INTEGER PRIMARY KEY AUTOINCREMENT,
    project_id       INTEGER REFERENCES projects(id) ON DELETE SET NULL,
    duration_minutes INTEGER NOT NULL,
    date             TEXT NOT NULL,
    created_at       TEXT DEFAULT (datetime('now', 'localtime'))
);
```

### tags 表
```sql
CREATE TABLE tags (
    id    INTEGER PRIMARY KEY AUTOINCREMENT,
    name  TEXT NOT NULL UNIQUE,
    color TEXT DEFAULT '#4A90D9'
);
```

### note_tags 关联表
```sql
CREATE TABLE note_tags (
    id         INTEGER PRIMARY KEY AUTOINCREMENT,
    project_id INTEGER NOT NULL REFERENCES projects(id) ON DELETE CASCADE,
    tag_id     INTEGER NOT NULL REFERENCES tags(id) ON DELETE CASCADE,
    UNIQUE(project_id, tag_id)
);
```

### app_index 表
```sql
CREATE TABLE app_index (
    id              INTEGER PRIMARY KEY AUTOINCREMENT,
    name            TEXT NOT NULL,
    path            TEXT NOT NULL,
    last_scanned_at TEXT DEFAULT (datetime('now', 'localtime'))
);
```

## 四、各模块技术方案

### 项目管理
- 语言识别：`utils/language_detector.py` 扫描目录扩展名，统计占比，映射到语言标签
- 编辑器启动：`subprocess.Popen([editor_path, project_path])` 
- 图标：QtAwesome 提供语言和编辑器图标

### 快速启动器
- 扫描：`utils/app_scanner.py` 遍历开始菜单、桌面、Program Files，收集 `.exe` 和 `.lnk`
- 匹配：`utils/fuzzy_matcher.py` 封装 `rapidfuzz.process.extract()`，返回 Top 10 结果
- 启动：`subprocess.Popen(app_path, shell=True)`
- 索引缓存：结果存入 `app_index` 表，支持手动刷新

### 学习计时器
- 计时：`QTimer` 每秒触发，倒计时逻辑在 Controller 层
- 图表：`matplotlib.figure.Figure` 嵌入 `FigureCanvasQTAgg`，绘制热力图和折线图
- 热力图：用 `matplotlib` 自绘日历格子，颜色映射学习时长

### 网络抓包
- 抓包线程：`QThread` 子线程运行 `scapy.sniff()`，通过 `pyqtSignal` 推送数据包到主线程
- 数据包解析：`pyshark` 解析详情，`scapy` 负责实时捕获
- 指定目标：`socket.gethostbyname()` 解析域名，scapy filter 参数过滤 IP
- 权限提示：启动时检测是否以管理员身份运行，否则弹出提示

### TODO 看板
- 拖拽：`QListWidget` 配合 `DragDropMode`，跨列拖拽触发状态更新
- 汇总视图：查询所有 `status != 'done'` 的任务，按项目分组展示

### Markdown 笔记
- 渲染：`QTextEdit` 配合 `setMarkdown()` 方法（PyQt5 5.14+ 支持）
- 导出：`QPrinter` 导出 PDF，`QTextEdit.toHtml()` 导出 HTML

## 五、主题方案

- 默认深色主题，通过 `QApplication.setStyleSheet()` 全局应用 QSS 样式表
- 主色调：深灰背景 `#1E1E1E`，强调色 `#4A90D9`，文字 `#D4D4D4`
- 支持浅色主题切换，QSS 文件分开存放（`assets/theme_dark.qss` / `assets/theme_light.qss`）

## 六、打包方案

使用 **PyInstaller** 打包为单个 `.exe` 文件，方便分发。

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --icon=assets/icon.ico main.py
```

注意事项：
- scapy 打包需要额外配置 hook
- matplotlib 字体资源需要手动包含
- 运行时仍需要 Npcap 驱动（不随 exe 打包）
